ScratchV 

ScratchV é uma ferramenta educacional que traduz programas desenvolvidos no Scratch para Assembly RISC-V, permitindo que estudantes compreendam a relação entre programação em blocos e programação de baixo nível. 

O objetivo do projeto é facilitar o ensino de Arquitetura de Computadores, Organização de Computadores e linguagens de montagem, conectando programação visual com a execução em arquitetura RISC-V.
